# Список ошибок и предложений для улучшения

### Ошибки:
1. Кнопка «Сохранить» не всегда работает при первом клике.
2. Приложение вылетает при попытке сохранить пустую транзакцию.

### Предложения:
1. Добавить опцию редактирования категории транзакции.
2. Добавить возможность группировки транзакций по неделям или месяцам в статистике.
