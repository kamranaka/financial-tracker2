Описание проекта
Financial Tracker 2 — это JavaFX-приложение для управления личными финансами. Оно помогает пользователям отслеживать свои доходы и расходы, планировать бюджет и анализировать свои финансовые потоки. Проект идеально подходит для использования студентами и людьми, которые хотят более эффективно управлять своими деньгами.

Основные функции:

CRUD операции с транзакциями: создание, редактирование, удаление и просмотр записей о доходах и расходах.

Фильтрация транзакций: возможность фильтрации данных по категориям, сумме, дате.

Добавление и редактирование категории расходов: возможность создавать категории для разных типов расходов (например, еда, транспорт, развлечения).

Отчеты и графики: визуализация данных с помощью графиков и диаграмм, чтобы пользователи могли отслеживать свои расходы и доходы.

Сохранение и загрузка данных в формате JSON: данные можно сохранять и загружать, что позволяет пользователю сохранять информацию при перезагрузке приложения.

Мини-статистика: отображение суммы расходов, доходов и общего баланса за выбранный период.

Установка и запуск
Шаг 1: Установка Java

Убедитесь, что у вас установлен Java JDK 17 или выше. Вы можете скачать JDK с официального сайта Oracle
.

Шаг 2: Клонирование репозитория

Для начала клонируйте репозиторий на свою машину:

git clone https://github.com/your-username/financial-tracker2.git

Шаг 3: Открытие проекта

Откройте проект в IntelliJ IDEA или другой IDE с поддержкой JavaFX.

Шаг 4: Добавление JavaFX в зависимости

Добавьте JavaFX в зависимости проекта. Если вы используете IntelliJ IDEA, вы можете добавить необходимые библиотеки через Project Structure.

Шаг 5: Запуск проекта

Запустите файл App.java как Java Application. Приложение откроется, и вы сможете начать использовать его для учета своих финансов.

Примеры использования

Создание транзакции: Нажмите кнопку «Добавить транзакцию», заполните поля и выберите категорию (доход или расход), затем нажмите «Сохранить».

Удаление транзакции: Выберите запись из списка и нажмите «Удалить». Подтвердите действие.

Фильтрация транзакций: Введите текст в поле поиска, чтобы отфильтровать транзакции по названию.

Просмотр графиков: Перейдите на вкладку «Статистика», чтобы увидеть графики и диаграммы своих расходов и доходов.
