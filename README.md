# Financial Tracker 2

Система для учета расходов и анализа финансов. Этот проект помогает пользователям отслеживать свои доходы и расходы, а также анализировать их финансовое состояние для более эффективного планирования бюджета и улучшения финансового положения.

 Описание проекта

Financial Tracker 2 — это приложение для личного учета финансов. Оно предоставляет пользователям возможность:
- Вести учет доходов и расходов.
- Просматривать отчет о расходах за выбранный период.
- Анализировать финансовое состояние с помощью диаграмм и графиков.
- Оценивать и планировать будущие расходы.

Проект предназначен для всех, кто хочет улучшить свои навыки в управлении личными финансами и разобраться в своих расходах и доходах.

Основные функции:
- Добавление доходов и расходов.
- Просмотр детализированных отчетов.
- Строительство графиков для наглядного представления данных.
- Анализ финансового состояния.
