# План тестирования для Financial Tracker 2

### Цели тестирования:
1. Проверка всех CRUD операций.
2. Проверка правильности отображения данных на графиках.
3. Проверка функционала фильтрации и поиска транзакций.
4. Оценка работы валидации и ошибок ввода.

### Среда тестирования:
- Операционные системы: Windows 10, macOS
- Инструменты: IntelliJ IDEA, Gson для работы с JSON
- Java 17, JavaFX

### Сценарии тестирования:
1. Создание и удаление транзакций
2. Фильтрация транзакций по категории
3. Сохранение и загрузка данных в формате JSON
4. Проверка отображения графиков
5. Валидация ввода данных (проверка ошибок)
